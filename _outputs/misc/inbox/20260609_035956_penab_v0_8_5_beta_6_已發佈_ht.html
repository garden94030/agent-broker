penab v0.8.5-beta.6 已發佈！

🔗 https://github.com/openabdev/openab/releases/tag/openab-0.8.5-beta.6

🚀 新功能
Secrets 管理機制 — 支援 AWS Secrets Manager + exec:// provider（#1042）
aws-sm:// URI：原生 AWS Secrets Manager 解析
exec:// URI：通用外部腳本支援
Fail-closed 設計：未解析的 secret 將阻止啟動
整合至 boot 流程（pre_boot hooks 之後執行）
完整文件、ADR 及 config-reference 更新

🐛 Bug 修復
Agent 回傳 -32603 但無錯誤細節時，顯示友善提示訊息（#1041, closes #1000）
新增 pending-decision reusable workflow，修復 needs-rebase label 衝突（#1040）
鎖定 kiro-cli 至 2.5.0 以避免 2.6.0 的 libasound.so.2 regression（#1043）
修正 hermes 安裝腳本中 /root/.local/share/uv 的 chmod guard

|-----|------|------|
| kiro | 2.6.0 → 鎖定 2.5.0 | libasound regression |
| codex | 0.137.0+acp 0.15.0 | 修復 output buffer resend + detach |
